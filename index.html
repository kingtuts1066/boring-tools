All tools

Every tool is a single HTML page, SEO-focused, and linkable.

  • Internet speed test Latency, download, upload
  • System info CPU threads, RAM estimate, GPU renderer (WebGL)
  • DNS lookup A/AAAA/MX/TXT/CNAME/NS via DNS-over-HTTPS
  • WebRTC leak test Check what IPs WebRTC can expose (VPN troubleshooting)
  • Browser benchmark JS compute + DOM + JSON + canvas (quick smoke test)

  • Loan repayment calculator Monthly payment, weekly/yearly costs, total interest + balance chart
