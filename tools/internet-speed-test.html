FAQ

Why is my speed different from my ISP plan?

Wi‑Fi quality, router hardware, VPNs, congestion, and the test server location can all reduce measured speed.

Does this test work on mobile?

Yes, but mobile networks can fluctuate rapidly. Try a few runs and look for a typical range.
